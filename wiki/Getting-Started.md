# Getting Started

This page explains how to set up your environment and run the LeetCode solutions in this repository.

---

## Prerequisites

- Python 3.x installed on your machine
- Git installed

---

## Clone the Repository

```bash
git clone https://github.com/hariharasudhan-29507/LEETCODE.git
cd LEETCODE
```

---

## Running a Solution

Each solution file is a standalone Python script. Navigate to the `Codes/` directory and run the file directly with Python.

```bash
python Codes/1.TwoSum.py
```

For files with spaces in the name, use quotes:

```bash
python "Codes/14.Longest Common Prefix.py"
```

---

## File Naming Convention

Solution files follow this pattern:

```
<problem_number>.<ProblemName>.py
```

Examples:

| File                                        | Problem Number | Problem Name               |
|---------------------------------------------|----------------|----------------------------|
| `1.TwoSum.py`                               | 1              | Two Sum                    |
| `9.Palindrome.py`                           | 9              | Palindrome Number          |
| `14.Longest Common Prefix.py`               | 14             | Longest Common Prefix      |
| `20.ValidParenthesis.py`                    | 20             | Valid Parentheses          |
| `22.GenerateParenthesis.py`                 | 22             | Generate Parentheses       |
| `26.RemoveDuplicatesFromSortedArray.py`     | 26             | Remove Duplicates          |
| `27.RemoveElement.py`                       | 27             | Remove Element             |
| `28.FindtheIndexoftheFirstOccurrenceinString.py` | 28        | Find Index of First Occurrence |
| `485.MaxConsecutiveOnes.py`                 | 485            | Max Consecutive Ones       |
| `1022.SumOfRootToLeafBinaryNumbers.py`      | 1022           | Sum of Root To Leaf Binary Numbers |
| `1470.ShuffleTheArray.py`                   | 1470           | Shuffle the Array          |
| `1929.ConcatenationOfArray.py`              | 1929           | Concatenation of Array     |
